# DOCX 檢閱者名稱整理

> **DOCX Reviewer Name Cleaner** · **DOCX 校閲者名の整理**

一頁式的小工具，讓你把 Word 檔（.docx）裡追蹤修訂（Track Changes）的檢閱者名稱批次改寫成你想要的名字。**不需要安裝任何軟體**、不需要上傳檔案到任何伺服器，打開網頁就能用。

介面支援**繁體中文 / English / 日本語**三種語言，右上角可隨時切換。

---

## 🧭 這個工具在解決什麼問題？

多人共同審閱的 Word 文件（例如律師事務所內部審過的契約）常常會遇到一個狀況：每個人 Word 上顯示的「檢閱者名稱」都不一樣，追蹤修訂上就會留下五花八門的署名。

在把檔案寄給客戶、對造或法院之前，通常會希望把這些名稱**統一成一個對外的單一名稱**（例如事務所名或對窗口的律師名），一方面避免把內部審閱分工的經過暴露給外部，另一方面也讓文件呈現得更一致、專業。

這個工具就是為了把那個流程變簡單：拖檔、改名、下載，幾秒鐘完成。

---

## 📥 如何下載（不需要會寫程式）

1. 打開這個 GitHub 頁面：
   **https://github.com/han0302-cyber/docx-reviewer-cleaner**
2. 在頁面檔案列表的**右上方**找到綠色的 **`Code`** 按鈕，點下去。
3. 在跳出的選單**最下方**選擇 **`Download ZIP`**。
4. 瀏覽器會下載一個 `docx-reviewer-cleaner-main.zip` 檔。
5. **解壓縮**這個 ZIP 檔（Mac 雙擊即可；Windows 按右鍵 → 解壓縮）。

解壓後你會看到一個資料夾，裡面有一個名叫 `index.html` 的檔案，這就是工具本體。

---

## 🚀 如何開始使用

1. 在解壓後的資料夾裡，**雙擊 `index.html`**。
2. 你預設的瀏覽器（Chrome、Safari、Edge、Firefox 都可以）就會自動把它打開。
3. 接下來就是操作畫面上的步驟：

   1. 把要處理的 `.docx` 檔案**拖到虛線框內**，或點一下虛線框選擇檔案。
   2. 頁面會自動列出檔案裡所有的檢閱者，以及每一位留下的修訂筆數。
   3. 在「**新名稱**」那一欄輸入你想把這位檢閱者改寫成的名字。如果某一位不想動，留空即可。
   4. 按右下角的「**套用並下載**」，就會下載一份新檔案，檔名是 `原檔名_renamed.docx`。
   5. 原始的檔案完全不會被動到，新檔就是處理後的結果。

完成。把新檔拿去寄信就可以了。

## ❓ 常見問題

**Q：需要安裝 Node.js 或其他套件嗎？**
不需要。整個工具只是一個 HTML 檔，雙擊就能用。

**Q：要先註冊 GitHub 帳號嗎？**
不需要。下載 ZIP 不需要登入。

**Q：我的電腦沒有網路，還能用嗎？**
可以。把 ZIP 下載下來一次之後，後續使用完全不需要網路。

**Q：處理過後的檔案跟原檔會有差異嗎？**
只有被你指定改名的那幾位檢閱者的 `w:author` 與 `w:initials` 會被改寫，其他內容、格式、追蹤修訂、註解、原本的審閱歷程都原封不動。

**Q：支援哪些瀏覽器？**
主流的現代瀏覽器（Chrome、Safari、Edge、Firefox）都可以。
